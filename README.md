
Moto Path Pro 🏍️

An AI-powered bike trip planner built using React, TypeScript, Vite, and Supabase.

Moto Path Pro helps riders plan, organize, and manage bike trips through a modern and responsive web application.

🚀 Features
🗺️ Bike trip planning and management
📍 Route and destination organization
⚡ Fast and responsive UI using Vite
🔐 Authentication-ready architecture
☁️ Backend integration with Supabase
📱 Responsive design for multiple devices
🧩 Modular and scalable project structure
🎨 Modern frontend design

⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/yourusername/moto-path-pro.git
2️⃣ Navigate to Project Folder
cd moto-path-pro
3️⃣ Install Dependencies
npm install
4️⃣ Configure Environment Variables

Create a .env file in the root directory:

VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
▶️ Run the Project

Start the development server:

npm run dev

The application will run at:

http://localhost:5173
